### 🎯 核心特色

- **🔄 实时监控** - 5秒间隔自动刷新股价数据，掌握每个交易时刻
- **📊 专业分析** - 详细数据展示，包含开高低收、今日盈亏、持仓盈亏、盈亏率分析、市值统计
- **🏷️ 多标签管理** - 自选股票支持多个标签，灵活分组管理

- **⭐ 自选股票** - 独立的股票关注列表，支持多标签分组和持仓高亮
- **🌐 全球市场** - 支持 A 股、美股、港股等主流市场股票代码格式

## 📡 数据来源与API

### 🎯 多层级数据策略

系统采用多数据源 + 自动容错机制，确保数据获取的稳定性和准确性：

```
股票代码 → [中国股票?] → [腾讯API] → [成功?] → 返回数据
                    │              │           │
                    ↓              ↓           ↓ 失败
                [Finnhub API] → [成功?] → [返回null] → 显示"-"
```

### 📊 数据源详情

| 优先级 | API名称 | 适用市场 | 数据完整度 | 更新频率 |
|:------:|----------|----------|------------|----------|
| 🥇 **1** | **腾讯API** | 🇨🇳 A股市场 | 完整（含开高低收） | 实时 |
| 🥈 **2** | **Finnhub API** | 🇺🇸 美股 / 🇭🇰 港股 | 基础数据 | 实时 |

### 🔄 容错机制

- **自动重试**: API失败后自动尝试下一个数据源
- **智能选择**: 根据股票类型选择最优数据源
- **真实数据**: 只显示真实API获取的数据，无法获取时明确提示
- **优雅失败**: 数据获取失败时显示"-"，不影响程序正常运行

## 🌟 特色功能

### 📈 分时数据后台采集

系统提供自动化的分时数据采集功能，为未来的分时图表和量化分析提供数据基础：

#### 🎯 核心特性
- **自动触发**: 进入持股或自选页面时自动启动后台采集
- **并发控制**: 最多10个goroutine并发执行，防止资源耗尽
- **智能更新**: 每分钟自动更新，仅在交易时间内采集（09:30-11:30, 13:00-15:00）
- **API降级**: 新浪财经（主） → 东方财富（备），自动切换提高可用性
- **永久保存**: JSON格式存储，按股票代码和日期组织，数据永久保留

#### 📁 数据存储结构
```
data/intraday/
├── SH600000/
│   ├── 20251202.json
│   └── 20251203.json
├── SZ000001/
│   └── 20251202.json
└── ...
```

#### 📊 数据格式示例
```json
{
  "code": "SH600000",
  "name": "浦发银行",
  "date": "20251202",
  "datapoints": [
    {"time": "09:31", "price": 8.52},
    {"time": "09:32", "price": 8.53}
  ],
  "updated_at": "2025-12-02 15:00:00"
}
```

#### 💡 使用说明
- 功能默认启用，无需配置
- 开启调试模式后按 `d` 键可查看采集日志
- 数据文件可用于未来的分时图表功能或导出分析

> 📝 **详细文档**: 查看 [分时数据采集功能文档](doc/issues/INTRADAY_FEATURE.md) 了解更多技术细节

### 🎨 智能配色系统

系统采用专业股票软件配色标准：

- 🔴 **红色**: 盈利/上涨数值 (> 0.001)
- 🟢 **绿色**: 亏损/下跌数值 (< -0.001)
- ⚪ **白色**: 持平数值 (≈ 0)，如 0% 涨幅、¥0 盈亏

### 🕰️ 实时更新机制

- **更新频率**: 每 5 秒自动刷新
- **智能缓存**: 避免频繁 API 请求
- **容错机制**: API 失败时自动降级处理
- **排序实时性**: 排序结果随股价实时更新

### 📊 精准计算

- **今日涨幅**: 基于昨收价计算 `(当前价 - 昨收价) / 昨收价 × 100%`
- **盈亏统计**: 实时计算今日盈亏、持仓盈亏、盈亏率
- **市值计算**: 现价 × 持股数量

### 🌍 国际化支持

- **完整多语言**: 支持中英文双语界面
- **动态切换**: 运行时可切换语言
- **文化适配**: 不同语言使用不同的颜色标准
- **配置持久**: 语言设置自动保存
